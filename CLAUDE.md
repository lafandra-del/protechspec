# Лендинг для услуг техспеца

> Личность, рабочий цикл и общие принципы — в глобальном `~/.claude/CLAUDE.md`.
> Здесь только то, что специфично для этого лендинга.

## Что это
Одностраничный сайт-визитка техспеца онлайн-школ.

## Стек
- HTML, CSS, JavaScript (без фреймворков)
- Один файл index.html со всем стилем внутри
- Без сборщиков, без npm, без зависимостей

## Стиль
- Тёмная тема
- Современный минимализм, без лишнего
- Один акцентный цвет
- Шрифты — системные или с Google Fonts
