# Партнерская программа

{% hint style="success" %}
Внимание-внимание 🎉

Оставляйте свои честные отзывы и получайте **промокод на 10 дней с полным функционалом Salebot.**&#x20;

{% endhint %}

{% hint style="warning" %} <mark style="color:red;">**Обращаем внимание!**</mark>

В Партнерской программе вправе участвовать ИП, юр. лица и самозанятые.

Партнерские выплаты можно использовать для оплаты новых или действующих проектов.

Выплаты по партнерской программе осуществляются в течение недели по Актам, заполненным в разделе "Партнерская программа".&#x20;

Если у вас остались вопросы по Актам, партнерской программе или иным документам, обращайтесь по электронному адресу - <documents@salebot.pro>
{% endhint %}

{% hint style="warning" %}
Обращаем внимание!

Выставлять счет в партнерском проекте или инициировать оплату нужно с аккаунта, который зарегистрирован по уникальной ссылке партнера.&#x20;

Если счет в проекте будет выставлен от аккаунта Партнера, который распространяет свою ссылку для регистрации на сайте или промокод для создания проекта, то вознаграждение не начисляется!
{% endhint %}

{% hint style="warning" %}
Партнерские выплаты осуществляются только для резидентов Российской Федерации.
{% endhint %}

{% hint style="danger" %}
Обращаем внимание: в случае, если Акт партнера сформирован с ошибкой или при его оплате произошла ошибка,  партнерское вознаграждение НЕ НАЧИСЛЯЕТСЯ.&#x20;

Например, если счет сформирован неверно: не указан номер счета в назначении платежа, то продление будет ручное. В таком случае выплаты по партнерской программе не осуществляются.

Также начисления не будет при переносе оплаты с одного проекта на другой.&#x20;
{% endhint %}

## Как стать партнером

Стать партнером просто:

1. Перейдите в раздел "Партнерская программа" (кликните на выпадающее меню рядом с тарифом)

<div data-with-frame="true"><figure><img src="/files/Rbs77jCIq2XhE3Q7uX3J" alt="" width="375"><figcaption><p>Рис. 2. Как перейти в раздел "Партнерская программа" в личном кабинете</p></figcaption></figure></div>

2\. Подключитесь к партнерской программе, нажав кнопку "Стать партнером". Настоятельно просим ознакомиться с условиями Публичной оферты, поскольку нажав кнопку "Стать партнером", вы принимаете условия партнерской программы:

<div data-with-frame="true"><figure><img src="/files/IbB5ZxtqDQ9a5iw4k7kg" alt=""><figcaption><p>Рис. 3. Как стать партнером</p></figcaption></figure></div>

3\. Скопируйте Вашу уникальную реферальную ссылку, доступную в этом разделе, и разместите ее в соц.сетях, на форумах, на своем сайте или блоге, отправьте друзьям и клиентам по Email, Viber, WhatsApp. Чем больше ссылок, тем больше шансов на подключение!

<div data-with-frame="true"><figure><img src="/files/FaXQFSWEREfPtyv4Y4v0" alt="" width="563"><figcaption><p>Рис. 4. Личный кабинет партнера</p></figcaption></figure></div>

{% hint style="success" %}
Важно!&#x20;

Реферальная ссылка является уникальной для каждого Партнера.
{% endhint %}

4\. Каждый зарегистрировавшийся по вашей ссылке попадает в список рефералов. А проекты, созданные вами или с использованием вашего персонального промокода, попадают в список реферальных проектов.

{% hint style="success" %}
Для получения персонального промокода напишите в тех. поддержку. В сообщении укажите почту вашего личного кабинета, к которому привязана партнерская программа и промокод. Промокод может быть на латинице или кириллице.
{% endhint %}

{% hint style="danger" %}
Важно!&#x20;

<mark style="color:red;">**Персональный промокод имеет приоритет выше**</mark>, чем партнерская ссылка.

Если Вы привлекли клиента с помощью партнерской ссылки, при этом Реферал создал проект с **использованием промокода другого партнера**, то реферальные бонусы начислятся тому, чей промокод был использован.&#x20;
{% endhint %}

**Самое приятное — Вывод средств**.

Накопленными средствами можно оплачивать подписку проектов (от 1 месяца подписки любого тарифа) и выводить <mark style="color:red;">**на расчетный счет ИП, юр. лица или на карту самозанятого**</mark>, начиная с 5000 рублей.&#x20;

Чтобы вывести средства с Партнерского счета (не является расчетным счетом), необходимо заполнить Акт партнера в разделе "Партнерская программа":

<div data-with-frame="true"><figure><img src="/files/Yz3xa22rwQisK4PDBP2i" alt=""><figcaption></figcaption></figure></div>

После того как вы заполнили Акт, который направляется в систему Salebot, вы получите запрошенные средства в течение недели.

{% hint style="danger" %}
*Сервис не производит начисления и выплат средств, полученных на основе ошибок в ПО, либо полученных обманным путем.*
{% endhint %}

***Рефералы*** - пользователи Salebot, впервые зарегистрированные в сервисе по вашей реферальной ссылке, с оплат которых начисляются партнерские проценты

***Реферальные проекты*** - проекты, за оплаты которых будут начисляться партнерские проценты. К ним относятся проекты:

* созданные с использованием вашего персонального промокода;
* ваши проекты при оплате подписки за проект другим пользователем Salebot со своего аккаунта.&#x20;

## Порядок начисления партнерских процентов

Партнерские проценты можно получать за оплату подписки реферальных проектов и с оплат ваших рефералов. Рассмотрим особенности начисления.

### *Начисления с оплат рефералов*

Пользователь впервые регистрируется в Salebot по вашей реферальной ссылке.

В таком случае система будет распознавать, что этот человек приглашен вами. Вам будет начисляться 10% с его оплат, если он создает проект бота, за который платит.

<mark style="color:orange;">**Исключение:**</mark>&#x20;

Если <mark style="color:green;">проект</mark> создан третьим лицом (например, разработчиком), а затем права на этот проект переданы человеку, который пришел по вашей партнерской ссылке, то партнерские <mark style="color:green;">10% с оплаты проекта</mark> будут начислены **тому, кто создал проект бота**.

{% hint style="danger" %}
*Сервис не производит начисления и выплаты средств, полученных на основе ошибок в ПО либо полученных обманным путем.*
{% endhint %}

{% hint style="warning" %}
Если **ваш реферал** создал проект с использованием **ЧУЖОГО** (не вашего) персонального **промокода**, партнерские выплаты будут **начислены владельцу (НЕ вам) ПРОМОКОДА**
{% endhint %}

### Начисления с реферальных проектов

Вы создали проект и передали заказчику. Далее заказчик сам оплачивает этот проект. *Партнерские проценты начисляются вам.*

Ваш персональный промокод был использован другим пользователем Salebot при создании проекта. *Партнерские проценты начисляются вам при оплате этого проекта.*

{% hint style="success" %}
**Партнерские проценты начисляются ТОЛЬКО при оплате через платежную систему сайта salebot.pro или по счёту, выставленному в разделе Оплата.**&#x20;

**Счет должен выставлять Реферал со своего аккаунта!**&#x20;

**Если счет будет выставлен с аккаунта Партнера, даже с указанием реквизитов реферала, бонусы не начисляются, потому что система фиксирует оплату за Партнером!**
{% endhint %}

{% hint style="danger" %}
**Партнерские&#x20;**<mark style="color:orange;">**проценты НЕ начисляются:**</mark>

* **если оплачиваете проекты с реферального счёта**
* **если оплачиваете прямым переводом на карту** (с карты на карту)
* **если счет выставлен НЕ на сайте**, а по запросу на почту
* **если оплачивается реферальный проект из своего аккаунта** (пример: создали проект для себя, использовали свой промокод и сами оплатили проект)
* **если Партнер сам выставил счет со своего аккаунта для Реферала или для себя** (например, Партнер создал проект для Реферала, настроил проект и выставил счет со своего аккаунта в разделе "Оплата" — оплата будет приписана Партнеру! <mark style="color:red;">**Нужно чтобы счет был выставлен с аккаунта Реферала!**</mark>)
  {% endhint %}

## Выплаты и списание партнёрских выплат&#x20;

Партнерские начисления не сгорают, у них нет срока действия.

### Оплатить подписку проекта

{% hint style="danger" %}
Оплатить с партнерского счета вы можете полный тариф подписки на срок от одного месяца.

Перерасчет реферальных средств в дни, часы, минуты подписки НЕ производится
{% endhint %}

Для оплаты подписки проекта напишите в техническую поддержку Salebot. Можно сразу предоставить всю необходимую информацию, чтобы специалисты поддержки оперативно обработали запрос:

1. Скриншот раздела "Партнерская программа" вашего личного кабинета, чтобы была видна почта аккаунта и сумма накопленных партнерских процентов
2. Почту вашего личного кабинета, к которому привязана партнерская программа.
3. Номер проекта с указание тарифа и всех необходимых дополнительных опций к тарифу.&#x20;

После обработки запроса специалист сообщит, что подписка в проекте активна. Средства будут списаны с партнерского счёта.&#x20;

### Вывести на расчётный счёт ИП или на карту самозанятого

Минимальная сумма выплаты - 5000 руб.

Вывести можно на расчетный счет ИП, юридического лица или самозанятого.&#x20;

Подробнее об условиях Партнерской реферальной программы читайте в [Публичной оферте.](https://salebot.pro/ooo_offer_partner.pdf)

При получении вознаграждения по Партнерской программе можно выбирать следующие коды ОКВЭД:

* 63.11 – Деятельность по обработке данных, предоставление услуг по размещению информации и связанная с этим деятельность; услуги в сфере обработки информации, ее размещению и сопутствующей деятельности;
* 62.09 – Деятельность, связанная с использованием вычислительной техники и информационных технологий, прочая;
* 82.99 – Деятельность по предоставлению прочих вспомогательных услуг для бизнеса, не включенная в другие группировки.

## Правила

Запрещается вести рекламу на прямые запросы на сайт.

{% hint style="danger" %}
В случае нарушения правил партнерской программы, рефералы будут откреплены.
{% endhint %}

{% hint style="danger" %}
Если администрация salebot.pro посчитает поведение партнеров порочащими честь платформы, то она имеет право аннулировать партнерские отчисления и открепить рефералов.&#x20;
{% endhint %}

{% hint style="danger" %}
Администрация salebot.pro имеет право пересматривать правила партнерской программы в одностороннем порядке.
{% endhint %}

{% hint style="danger" %}
Salebot вправе лишить статуса партнера и прекратить выплаты бонусов, если откроется, что Партнер не является:

1. представителем юридического лица, от имени которого Партнер участвует в партнерской программе;&#x20;
2. &#x20;индивидуальным предпринимателем;
3. самозанятым лицом.
   {% endhint %}
