# WABA\*

Работа официального Whatsapp\* не отличается от работы "серого". То есть, созданный бот для "серого", будет работать аналогично в белом.

{% hint style="danger" %}
Whatsapp Busniess Api, Whatsapp принадлежат Meta inc., деятельность которой признана экстремистской и запрещена!
{% endhint %}

## Шаблоны сообщений для Whatsapp\* Business Api

При использовании Whatsapp\* Business Api пользователи не могут писать клиентам в любое время суток и направлять неограниченное количество сообщений при создании бота, аналогично серому Whatsapp\*, за пределами 24-часового окна обслуживания клиентов.

Вне 24-часового окна могут использоваться только шаблоны сообщений WhatsApp\* (или шаблонные сообщения WhatsApp\*), в течение которого общение происходит бесплатно в рамках выбранной категории.&#x20;

Данный функционал используется в Whatsapp\* для предотвращения спам-рассылок, ограничивая количество допустимых сообщений в определенный временной интервал, а также с присвоением категории шаблонам.&#x20;

Существуют следующие категории сообщений, установленные для Whatsapp\* Business Api компанией Meta\*:

1. Полезные шаблоны (содержат конкретную информацию о транзакциях, счетах, оплатах и пр.);
2. Шаблоны для маркетинга (информация о компании, организации, стратегиях, рекламе и продвижению);
3. Шаблоны для верификации и аутентификации.&#x20;

{% hint style="success" %}
Шаблоны, созданные для WABA\*, подвергаются модерации. \
Как создать шаблон для Whatsapp\* Business Api, читайте далее
{% endhint %}

{% hint style="danger" %}
\*Деятельность компании признана экстремисткой и запрещена на территории Российской Федерации.
{% endhint %}

### Основные причины отклонения шаблонов

Достаточное количество шаблонов не проходят модерацию в Whastapp\* BA, и на это есть ряд причин:

1. Слишком много рекламы в шаблоне.&#x20;

WABA\* отвергает шаблоны, переполненные рекламой и рекламными интеграциями. Если модераторы или нейросеть сервиса установит превышающее количество рекламы над основным текстом сообщения, шаблон заблокируют.

2. Нарушение товарной политики.

Все запрещенные наркотические и психотропные вещества, алкоголь, табачная продукция, услуги интимного характера и прочее: если в вашем сообщении будет хоть малейший намек на данные товары или услуги, уничижающее человеческое достоинство, шаблон не пропустят.

3. Недосказанность/двусмысленность/ложь

Если модераторы WABA\* каким-либо образом обнаружат в вашем сообщении скрытый подтекст или информацию, не соответствующую действительности, шаблон не пропустят.

Если нейросеть WABA\*, а также модераторы обнаружат в вашем шаблоне недостаток информации о компании, о сервисе, то шаблон не пройдет модерацию.

4. Сбор конфиденциальной информации.

Если ваш шаблон направлен на получение конфиденциальной информации пользователя (паспортные данные, номер банковских карт; личные серийные номера иных документов; адрес проживания и т.п.), то шаблон не пройдет модерацию.

5. Оскорбление пользователей

Если модераторы WABA\* обнаружат в вашем сообщение информацию, которая может оскорбить кого-либо из пользователей (по расе, национальности, половому признаку, идентичности - критериев много), то шаблон не пройдет модерацию.

6. Без объяснения причины.&#x20;

Модераторы WABA\* оставляют за собой право отклонять шаблоны без объяснения причины.&#x20;

### ​Какие есть особенности кнопок в шаблонах

Существует два типа предустановленных кнопок в шаблонных сообщениях:

**Call-To-Action** — это ссылки или номер телефона (рядом с самим текстом кнопки будет иконка ссылки или телефонной трубки);

**Quick reply** — это обычные текстовые кнопки, ограничение — 20 символов.

Кнопок типа Quick Reply в одном шаблоне может быть 3 шт., Call-to-Action — 2 шт., при этом нельзя сделать две ссылки или два номера.

Кнопки не могут содержать переменные, абзацы, эмоджи или символы форматирования.

При нажатии на кнопку Quick Reply, отправляется ее текст ответным сообщением.

На кнопку Call-to-Action можно прикрепить ссылку на файл. Кнопка Call-to-Action со звонком переносит в стандартное меню звонка телефона, с десктопа появится окно о том, что нельзя делать звонки в WhatsApp Web.

### Как проходит модерация шаблонов

Каждый шаблон после создания отправляется на модерацию. Обычно модерация занимает около 8 часов. Максимальное время модерации 48 часов.

Статус модерации отображается в списке шаблонов.

### Шаблоны, которые с большой вероятностью пройдут модерацию:

Курсивный текст может меняться

Здравствуйте, мы не смогли до Вас дозвониться. Вам удобно будет продолжить общение *здесь*?

Здравствуйте, Вы оставили обращение на сайте и просили с Вами связаться.(может меняться)

### Шаблоны, которые не пройдут модерацию:

любой рекламный текст

### Как проверить попадание в 24-часовое окно

Вы можете узнать сколько дней прошло с момента последней реплики клиента. Для необходимо использовать в условии функцию **days\_from\_last\_message()**

#### Пример проверка попадания в 24-часовое окно:

days\_from\_last\_message() == 0

## Ограничения неверифицированных аккаунтов

Для неверифицированных аккаунтов на Фейсбук<mark style="color:red;">\*</mark> есть ограничения: \
50 шаблонных сообщений/день(включены в ежемесячную плату). \
По входящему трафику ограничений нет.&#x20;

Чтобы снять ограничения, необходимо пройти верификацию компании, которая длится от 2-3 недель.

{% hint style="danger" %}
\*Принадлежит компании Meta, запрещенной в России и признанной экстремисткой.
