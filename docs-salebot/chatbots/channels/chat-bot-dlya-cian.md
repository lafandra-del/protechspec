# Чат-бот для Циан

Интеграция с популярными платформами позволяет значительно упростить работу с бизнес-процессами и автоматизировать рутинные задачи.&#x20;

**Salebot** — мощный инструмент для автоматизации общения с клиентами, который теперь поддерживает интеграцию с платформой для недвижимости в России — **Циан**.&#x20;

<div data-with-frame="true"><figure><img src="/files/4xVhKlBdPuDhxPfAuGnR" alt="" width="299"><figcaption></figcaption></figure></div>

Благодаря этому решению, пользователи могут эффективно управлять объявлениями, минимизируя ручной труд.

## Где взять ID и ключ

Перед подключением к Сейлбот интеграции, вам необходимо перейти на сервис Циан и войти в свой аккаунт:

<div data-with-frame="true"><figure><img src="/files/24aUwbM0cgrtkqzpJ6UT" alt=""><figcaption></figcaption></figure></div>

Здесь вы сразу найдете ID, который в последующем понадобится для подключения к Сейлбот:

<div data-with-frame="true"><figure><img src="/files/9iGXfxsXkyON5Q9QIYAP" alt="" width="501"><figcaption></figcaption></figure></div>

Также для работы с API необходимо получить <mark style="color:green;">**ключ авторизации**</mark>.&#x20;

Для получения ключа нужно написать письмо на <mark style="color:red;">**<import@cian.ru>**</mark> с темой "<mark style="color:red;">**ACCESS KEY**</mark>" и названием агентства, которое вы представляете.&#x20;

<div data-with-frame="true"><figure><img src="/files/JnM4EReUN3YIdgjNJKrZ" alt="" width="563"><figcaption></figcaption></figure></div>

Менеджер уточнит подробности и пришлет ACCESS KEY. Для каждой учетной записи на ЦИАН выдается отдельный ключ авторизации.

**Пример ключа:** fzKicHdOjJtJoSjPjKJV.J26dDJ7JlqYWDKzK2d3WXSBRjPkZxNEl6PUd6gR.lQggC7RFJBb:zknNontZ1VK8D2ikJ17U8nSKor24InJ

После того, как вы получили ключ и ID для своего аккаунта/организации, можно переходить в Сейлбот

## Подключение к Salebot

В Сейлботе вам необходимо перейти в раздел "Каналы":

<div data-with-frame="true"><figure><img src="/files/5O9lec0ryk94v2062oEu" alt="" width="449"><figcaption></figcaption></figure></div>

В разделе "Каналы" необходимо найти кнопку "Циан" для подключения интеграции:

<div data-with-frame="true"><figure><img src="/files/gpa4FUbYTRaZb2t6sb5W" alt=""><figcaption></figcaption></figure></div>

Нажмите на кнопку, чтобы открылось модальное окошко для подключения интеграции:

<div data-with-frame="true"><figure><img src="/files/ETDWcDpLQqRgmqs2zWEE" alt=""><figcaption></figcaption></figure></div>

Далее введите в пустые поля ID и уникальный ключ авторизации, который вы получили от менеджера Циан:

<div data-with-frame="true"><figure><img src="/files/rflSTQ1jvAAzWN2to92k" alt=""><figcaption></figcaption></figure></div>

Далее нажмите на кнопку "Готово", чтобы канал был подключен к проекту:

<div data-with-frame="true"><figure><img src="/files/bfi1Jr5xSPYk9zMHcPHf" alt=""><figcaption></figcaption></figure></div>

### Возможности подключенного бота

Благодаря подключенному боту в Циане, вы можете настроить воронку для автоматических ответов вашим клиентам, причем в диалоге с клиентом вы всегда увидете, по какому именно объекту недвижимости обратился клиент:

<div data-with-frame="true"><figure><img src="/files/TsxO4YLpxLOpektyYZS7" alt=""><figcaption></figcaption></figure></div>

Причем вы увидите:

1. Адрес объекта:

<div data-with-frame="true"><figure><img src="/files/WDzzGTablyIzqtthOkw7" alt="" width="560"><figcaption></figcaption></figure></div>

2. Уникальную ссылку на объект:

<div data-with-frame="true"><figure><img src="/files/i0ySwTpVHCCyzJl5CHZO" alt="" width="561"><figcaption></figcaption></figure></div>

3. Уникальные ссылки на фото объекта:

<div data-with-frame="true"><figure><img src="/files/siZj2UVVU6sSH1VFO3U9" alt="" width="563"><figcaption></figcaption></figure></div>

4. Значения объекта недвижимости в виде суммы аренды и характеристики объекта, причем все данные подтянутся автоматически из Циана:

<div data-with-frame="true"><figure><img src="/files/1AJb0ia99UWyigdB4v88" alt="" width="554"><figcaption></figcaption></figure></div>

Причем ваша сделка может быть сразу отражена в CRM-системе, где вы сможете отслеживать количество полученных и обработанных заявок:

<div data-with-frame="true"><figure><img src="/files/Xo8DqxK2HJ12M4DuSn2I" alt="" width="375"><figcaption></figcaption></figure></div>

{% hint style="success" %}
Таким образом, подключенный бот Циана к Сейлбот поможет вам видеть всех клиентов в едином диалоговом окне, также отслеживать и видеть, какой именно объект недвижимости заинтересовал вашего клиента, а также создавать сделки в CRM.&#x20;
{% endhint %}
