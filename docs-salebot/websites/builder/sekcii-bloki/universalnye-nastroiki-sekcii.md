# Универсальные настройки секций

Настройки шрифтов, фона и отображения универсальны для каждого блока: то есть каждая секция сайта отличается между собой контентом, однако настройка фона, шрифтов или отображения практически идентичны.

Чтобы найти универсальные настройки, наведите на необходимый блок курсором и найдите кнопку "Настройки":

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcJxqG2d5boBhx6OaqYx_oFcVnGDv8euU_6dw-hagy3wD7y4NKIEZESsHZoEN5qFsnGOnge9nX2IsUbouzljon83tuRIf8yza32EqJu_u3ku7nFlHb-uQVX2UcA8MsKF-1ORne9aQ?key=wvItsTVZeN1qx7iWn8-XSKvr" alt=""><figcaption></figcaption></figure>

Кликнув по кнопке “Настройки”, откроется правое меню с универсальными настройками секции:

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdXk5K13D4mcs7gm2QPNDD7LcSrf3GH70bjk0EEz-H0_fTqCZ12WYXcm6Pa6WAgfWGP1XbZXfBh5k9aXEE39wVZjl7EWZO9HHvoDrRdw_bzMGrp4-b--9XLk2lKUAj7CmI2Yjli?key=wvItsTVZeN1qx7iWn8-XSKvr" alt=""><figcaption></figcaption></figure>
