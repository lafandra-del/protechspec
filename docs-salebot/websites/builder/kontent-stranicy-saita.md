# Контент страницы сайта

Любой сайт (многостраничный или лендинг) состоит из страниц(ы), которая содержит в себе определенный контент.

<figure><img src="/files/d2a7uzNJBCSMGUc4pZPk" alt=""><figcaption></figcaption></figure>

Контент страницы состоит из секций (блоков), которые помогают наполнить ваш сайт различными интересными элементами интерфейса: тексты, изображения, кнопки, форма сбора заявок, виджеты, карта вашего положения и многое другое.&#x20;

{% hint style="success" %}
В конструкторе Сейлбот существуют различные типы блоков, где каждый блок (секция) создается на сайте с собственными настройками:

* [Универсальный блок](broken://pages/JJ6ATiOGjG9qZKXQtXfI) (зеро блок): блок без предустановленных настроек - можно разместить любой контент и самые разные элементы сайта по своему уникальному дизайну, при этом элементы могут быть расположены в любом количестве и в любом порядке, в отличие от иных блоков (секций) сайта. Зеро блок может выступать в качестве любой секции сайта, будь то меню, футер, форма сбора заявок, галереи и т.п.
* [Обложка:](broken://pages/sMF3lVbjwVwwvAelilRq) блок для добавления изображение и видео с заголовком.\
  Традиционно блок обложки используется  в качестве визитной карточки сайта, с которым первоочередно знакомятся посетители вашего сайта.
* [Заголовок и текст](broken://pages/dW4g0qINOoCqxI8pfqOv): текстовый блок с заголовком.&#x20;
* [Меню: ](broken://pages/1UThpTPciQsbPECtlvuK)для создания футера с кнопками в начале сайта (можно сделать плавающее меню).&#x20;
* [Онлайн-запись:](broken://pages/nRFzu0WTeKaztnvBy4Vw#vidzhet-onlain-zapisi-na-sait) для добавления формы онлайн-записи для клиентов (для уже настроенных филиалов) с автоматическим подключением к SalebotCRM.
* [Этапы](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-etapy) для создания уникального дизайна сайта с хронологией событий.
* [Аккордеон](broken://pages/7plT2EyILK8fA1leYtRZ): для добавления информации или данных на сайт в вопросно-ответной форме.
* [HTML](broken://pages/dW4g0qINOoCqxI8pfqOv): блок позволяет добавить на страницу HTML и CSS коды
* [Форма](broken://pages/YOzidl4HYJaWmkPXnuYY): блок для добавления кнопок мессенджеров, вопросов, политики конфиденциальности.&#x20;
* [Квиз формы](broken://pages/SLPSyrJDEiPAXVR2o8Zx): блок для создания постраничных опросов (квизов) на сайте.&#x20;
* [Медиа](broken://pages/dW4g0qINOoCqxI8pfqOv): блок для добавления изображений или видео
* [Кнопки](broken://pages/S4na9bSPZM4yI29COt4q): блок для добавления кнопок с функцией ссылки, телефона, email, оплаты
* [Цитата](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-citata): блок для добавления цитаты
* [Список:](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-spisok) блок с возможностью вставить нумерованные и маркированные списки
* [Выпадающий текст](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-vypadayushii-tekst): блок для добавления выпадающего текста
* [Галерея](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-galereya): блок для добавления галереи изображений.&#x20;
* [Pop-up](broken://pages/hxO6OwzOdQPIjJgFHJXc): для добавления всплывающего окна на сайте
* [Настраиваемые карточки](broken://pages/T6BXPfpfVLofCBGOTywB): поможет добавить отзывы клиентов или информацию о товаре.
* [Карточки курсов](broken://pages/FBlzIJMArVfogHlHjSGo) понадобится для продажи курсы прямо с сайта: вам не нужно самим собирать и настраивать карточку, данные подтягиваются из раздела "Онлайн-курсы".&#x20;
* [Список уроков](broken://pages/FBlzIJMArVfogHlHjSGo#kak-dobavit-blok-spiski-urokov): достаточно выбрать курс в настройках блока и все уроки этого курса отобразятся на сайте
* [Таймер](broken://pages/dW4g0qINOoCqxI8pfqOv#sekciya-taimer): блок для добавления таймера обратного отсчета.
* [Таблицы](broken://pages/RMsrn872SIjkuWZRKZiG): для добавления структурированной информации на сайт в виде таблиц.
* [Cookie](broken://pages/ZVwYfWbdbPd0W7xpi0Fs): блок для прозрачной работы сайта для пользователей, который показывает, что вы честно информируете, какие данные сохраняются с сайта и как они помогают улучшить взаимодействие вас и ваших пользователей.
  {% endhint %}

Чтобы наполнить страницу сайта контентом из секций, нужно перейти в режим редактирования страницы, что можно сделать следующим образом:

1. Многостраничный сайт

Для перехода в режим редактирования страницы сайта нажмите на плашку нужного сайта для перехода в его настройки:

<figure><img src="/files/EFtaBTg1imyCmoQqNLVp" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
В случае, если вы только создали новый многостраничный сайт, Вам никуда переходить не нужно, поскольку вы уже будете находиться во вкладке с настройками и страницами сайта.
{% endhint %}

Далее нажмите на нужную страницу сайта:

<figure><img src="/files/wYTOjzR7U5qqJ2fN0X48" alt=""><figcaption></figcaption></figure>

Тогда вам откроется форма редактирования страницы, где можно добавить/удалить блоки (секции) сайта, поменять шрифты, фон страницы и пр.:

<figure><img src="/files/dReFSMYslhrhqMysqi6h" alt=""><figcaption></figcaption></figure>

2. Лендинг

При создании одностраничного сайта вы сразу перейдете в режим редактирования страницы сайта:

<figure><img src="/files/KrGRsLmig1k72BGKpocc" alt=""><figcaption></figcaption></figure>

В случае, если необходимо отредактировать уже существующую страницу, то перейдите в раздел "Сайты", найдите необходимый лендинг (он будет помечен как "Лендинг" на плашке) и нажмите на него:

<figure><img src="/files/XXvQzgHJpTGYTSha70jo" alt="" width="563"><figcaption></figcaption></figure>

Далее вы перейдете в настройки и основную информацию о сайте. Здесь необходимо кликнуть по кнопке "Редактировать":

<figure><img src="/files/9x2g6qd7jMBT6Xw6JTzQ" alt=""><figcaption></figcaption></figure>

Аналогично вы сможете перейти в режим редактирования страницы с формой, опросом и квизом. При этом в общем разделе "Сайты" вы сможете увидеть, к какому типу относится тот или иной сайт:

<figure><img src="/files/uhaaiLYbekdPdoq6cjhH" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
**Горячие клавиши для входа в полноэкранный режим редактирования:**

`Ctrl+shift+X` или `Cmd+shit+X`
{% endhint %}

## Как добавить новую секцию (блок)

Чтобы добавить новый блок на странице, наведите на рабочую область страницы сайта и найдите кнопку "+":

<figure><img src="/files/6rGtRDRqPr4R6NwobSxg" alt=""><figcaption></figcaption></figure>

При клике на кнопку "+" откроется правое меню с выбором блока, который нужно добавить на страницу:

<figure><img src="/files/P487B7DZxgq4SSPZpFiA" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Подробно о настройке каждого блока (секции) сайта рассказали в одноименном разделе "[Секции (блоки)](broken://pages/9VvYi1sRwNdMmQhsrk3a)".
{% endhint %}

## Как удалить секцию (блок)

Чтобы удалить ненужный блок, наведите на него и найдите соответствующую кнопку:

<figure><img src="/files/sQk7qMTzhpASfMsE9GIS" alt=""><figcaption></figcaption></figure>

Нажмите на кнопку "удалить", тогда секция будет безвозвратно удалена.&#x20;

{% hint style="warning" %}
Важно!&#x20;

Удаленную секцию (блок) нельзя восстановить!
{% endhint %}

## Как скрыть секцию (блок)

В случае, если вы пока не хотите удалять секцию (блок) или нужно скрыть ее на какое-то время, то наведите на блок и найдите кнопку "выкл":

<figure><img src="/files/BrwQq2YhRXmNcY4gon9T" alt=""><figcaption></figcaption></figure>

При клике на кнопку секция будет скрыта:

<figure><img src="/files/qAsNaAOOyTmHVYWyoK2A" alt=""><figcaption></figcaption></figure>

Чтобы вернуть секцию, нажмите эту же кнопку.

## Как скопировать секцию

Чтобы скопировать секцию (например, для настройки разного [диапазона видимости на устройствах](#diapazon-vidimosti)), вам понадобится скопировать секцию, добавив ее на эту же страницу.&#x20;

Чтобы скопировать секцию, наведите на нее мышью и найдите кнопку для копирования:

<figure><img src="/files/4XyBafHd9QmneWcHnXII" alt=""><figcaption></figcaption></figure>

При клике на кнопку "Скопировать" секция будет добавлена ниже.

## Как редактировать секцию (блок)

Чтобы перейти в режим редактирования секции, наведите на нее мышью и кликните на кнопку "Редактировать":

<figure><img src="/files/yQEXUtJPIdjdaqeyiaut" alt=""><figcaption></figcaption></figure>

В зависимости от типа блока:

1. Вам откроется меню для редактирования контента блока справа (для всех блоков, кроме универсального):

<figure><img src="/files/3zfy4oKxIdpbWSk4ap2L" alt=""><figcaption></figcaption></figure>

2. Вам откроется страница с редактированием универсального блока (зеро блока) в отдельном окне:

<figure><img src="/files/BF0GZOLBvXiRbn3qevkQ" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Чтобы знать, как отредактировать любой блок и для чего нужны какие-либо функции и настройки, ознакомьтесь с разделом "[Секции (блоки)](broken://pages/9VvYi1sRwNdMmQhsrk3a)".
{% endhint %}

## Настройки секции (блока)

<figure><img src="/files/APgWUGMvHvNPwongAF9C" alt=""><figcaption></figcaption></figure>

Если сам контент секции редактируется при клике на кнопку "Редактировать", то остальные настройки (шрифтов, положения элементов блока, отображение секции и т.п.) осуществляются при клике на кнопку "Настройки" (для всех блоков, кроме универсального):

<figure><img src="/files/L6Lw8C1JYcVxh9x2V2xm" alt=""><figcaption></figcaption></figure>

Настройки шрифтов, фона и отображения являются универсальными настройками — это значит что такие настройки схожи между собой в разных секциях (блоках).

{% hint style="success" %}
Подробнее об универсальных настройках шрифтов, фона и контента рассказали в [одноименной статье в разделе "Секции (блоки)](broken://pages/3znyClBeT2ARaTjeA1OU)".
{% endhint %}

#### Настройка отображения

С помощью данного меню можно настроить отображение данного блока страницы и подписной ВК. А также сделать оптимизацию под любой тип устройств: настроить видимость блока в зависимости от размера экрана устройства.

<figure><img src="/files/4sp8H7Ufr5yB99evefvx" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="warning" %}
У подписной страницы <mark style="color:orange;">**ВКонтакте**</mark> фиксированные размеры, поэтому параметр <mark style="color:orange;">**Ширина секции и Ширина контента работать НЕ БУДУТ**</mark> (значения не применятся).
{% endhint %}

#### Диапазон видимости

В настройках отображения секции (блока) есть возможность указать для устройств при каком размере экрана будет отображаться текущая секция(блок).

<figure><img src="/files/0APgJU7q10FgCelAHH2W" alt=""><figcaption></figcaption></figure>

Таким образом можно для мобильных устройств сделать более адаптированную версию контента данной секции (блока):

<div><figure><img src="/files/9OswC2kZWY4FwndCSNzi" alt=""><figcaption></figcaption></figure> <figure><img src="/files/cN8pLQtEhKqmkpv0wJsr" alt=""><figcaption></figcaption></figure></div>

Таким образом, можно создать одну секцию, скопировать ее несколько раз и адаптировать уже готовые секции под различные размеры устройств.

Пример: одинаковые секции созданы на странице для разных устройств. При этом, первая секция отображается только для устройств, размер экрана которых не превышает 960 px.

## Как встроить кнопки к Вам на сайт

Чтобы интегрировать на ваш сайт форму с кнопками на мессенджеры (боты), нужно создать сайт или лендинг с формой или сайт с типом "Форма" на Salebot. После создания сайта у вас появится кнопка, как на картинке ниже.
