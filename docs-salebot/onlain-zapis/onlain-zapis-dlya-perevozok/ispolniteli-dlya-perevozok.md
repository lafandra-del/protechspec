# Исполнители для перевозок

## Как создать исполнителя для перевозок

После настройки вкладки с услугами перейдите во вкладку "Исполнители":

<figure><img src="/files/pNVvDDRiDgr8pR3ix6PO" alt=""><figcaption></figcaption></figure>

Здесь необходимо создать исполнителя: для этого нажмите на одноименную кнопку, после чего откроется модальное окошко с добавлением исполнителя:

<figure><img src="/files/htH4Czm6tGEJYISmKd8x" alt=""><figcaption></figcaption></figure>

Выберите тип сотрудника:

1. Рейсовый автобус

<figure><img src="/files/soB8A1rKPAHPBngSSbfW" alt=""><figcaption></figcaption></figure>

Обязательное поле — Название рейса:

<figure><img src="/files/DZwvVjxFEDVOZOg6ql5s" alt=""><figcaption></figcaption></figure>

Затем при необходимости пропишите описание рейса и добавьте изображение:

<figure><img src="/files/wvkcToNx2F6O8bxukHn8" alt=""><figcaption></figcaption></figure>

Теперь для исполнителя типа "Рейсовый автобус" можно назначить услуги:

<figure><img src="/files/0BbG2zyPOepgflLonw5t" alt=""><figcaption></figcaption></figure>

Вы можете создать любое необходимое количество рейсовых автобусах в разделе "Исполнители".

При клике на исполнителя откроются настройки:

1. Основная информация — для указания информации об автобусе (автомобиле)
2. Услуги — для назначения услуг;
3. Рабочий график — для установки времени и даты поездок.

## Основная информация

В разделе "Основная информация" можно изменить / добавить:

1. Название рейса;
2. Описание рейса;
3. Марку автобуса;
4. Гаражный номер;
5. Госномер;
6. ИП (АТП);
7. ИНН;
8. Условия приобретения билетов;
9. Изображение рейса (автобуса / автомобиля).

<figure><img src="/files/1GGCPd6Ju701yHnVHfi0" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
ИП — индивидуальный предприниматель.

АТП — автотранспортное предприятие.

В поле ИП (АТП) можно указать название вашего ИП или АТП.
{% endhint %}

Далее нажмите "Сохранить", чтобы информация была сохранения в разделе:

<figure><img src="/files/vH6FLKbzYGJnO3v6Avma" alt="" width="563"><figcaption></figcaption></figure>

## Назначить рейс

Чтобы назначить рейсу определенные маршруты, которые вы прописали в разделе "Услуги", перейдите во вторую вкладку настроек исполнителя:

<figure><img src="/files/TCz77IziaOpUvnrhCK4V" alt=""><figcaption></figcaption></figure>

Далее нажмите кнопку "Назначить услуги":

<figure><img src="/files/HLNNkk7sTiFKGIGDBran" alt=""><figcaption></figcaption></figure>

Откроется окно, в котором нужно выбрать, какие рейсы будет выполнять данный автобус:

<figure><img src="/files/0UXAhZOAFWZucMZgNYxD" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Можно назначить все рейсы, одну из категорий или один из рейсов.
{% endhint %}

<figure><img src="/files/lT60dcns61TQF0pTqFjx" alt=""><figcaption></figcaption></figure>

Далее нажмите "Сохранить". Тогда данному исполнителю типа "Рейсовый автобус" будут назначены выбранные вами рейсы.

<figure><img src="/files/O4Xmzh4h5LHyD28n2bJI" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Чтобы изменить рейсы для транспорта, кликните по кнопке "Назначить услуги" и выберите необходимые рейсы.
{% endhint %}

## Рабочий график

Чтобы назначить график для перевозок перейдите в раздел "Рабочий график":

<figure><img src="/files/AyS072y9dpb0msDWv4WA" alt=""><figcaption></figcaption></figure>

Здесь вы увидите шаблон для заполнения графика, где можно выставить:

1. Расписание:&#x20;

<figure><img src="/files/XDIK7VjktD65CIsOwO78" alt="" width="563"><figcaption></figcaption></figure>

Можно выставить, по каким дням будет осуществлять перевозки данный автобус (автомобиль).

2. Дату, с которой должен быть выставлен график:

<figure><img src="/files/lWChr3CJjP0otu6WxohQ" alt="" width="563"><figcaption></figcaption></figure>

3. Время работы:

<figure><img src="/files/wcCS6vnLk0AfdDNFBK4i" alt=""><figcaption></figcaption></figure>

В поле со временем работы нажмите на "Редактировать", чтобы открылся правый бар для настройки графика рейса.

Нажмите "Добавит рейс", тогда добавится плашка для настройки времени:

<figure><img src="/files/W7Iw2u0D4RES8hliLr1g" alt=""><figcaption></figcaption></figure>

Выберите тип рейса и время отправления (время прибытия расчитается автоматически):

<figure><img src="/files/lSTuQxCrDbA8GIekREaJ" alt="" width="563"><figcaption></figcaption></figure>

Далее выберите на какое количество недель нужно выставить график:

<figure><img src="/files/1JOqyLUn5LV1vJS0z0hh" alt="" width="549"><figcaption></figcaption></figure>

Затем нажмите "Заполнить и сохранить":

<figure><img src="/files/Dpl9OG5tsySApQmhFsgQ" alt="" width="563"><figcaption></figcaption></figure>

После того, как вы нажали на кнопку, чтобы заполнить график, будет создан утвержденный график:

<figure><img src="/files/GhY0Qc2OJoQEvlupnfgS" alt=""><figcaption></figcaption></figure>

В утвержденном графике можно увидеть:

1. Время начала поездки;
2. Время окончания поездки;
3. Количество пассажиров, забронировавших проезд;
4. Можно выгрузить путевой лист с информацией о пассажирах и их количестве;

Также можно удалить рейс при необходимости.
