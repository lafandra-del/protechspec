# Настройки рейса

Чтобы ваши клиенты могли бронировать места на рейсе, необходимо добавить различные услуги.&#x20;

Каждая услуга относится к определенной категории, поэтому для начала необходимо создать категорию услуг. После создания филиала вы увидите кнопку "Создать категорию":

<figure><img src="/files/BWeknj1n4mbJPjp0jP36" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Без создания категории, вы не сможете создать услугу.&#x20;
{% endhint %}

## Создание категорий услуг

Кликните на кнопку "**Создать категорию**", расположенной в середине экрана, как показано выше. После нажатия откроется окно для добавления новой категории. В поле "Название категории" выберите наименование категории, удобное для группировки рейсов

<figure><img src="/files/ObhszPKZa0lh3JpNm3ei" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Например, ночные рейсы, дневные рейсы и т.д.
{% endhint %}

После заполнения поля "Название категории" нажмите на кнопку "Сохранить", тогда категория будет добавлена в разделе услуги.

<figure><img src="/files/ts8h3OjQ8jm5RJjzpAhz" alt=""><figcaption></figcaption></figure>

Вы можете добавить любое количество категорий услуг, для этого найдите кнопку справа экрана "Добавить категорию":

<figure><img src="/files/Qcly3dadCU3rIIbi1Ogf" alt=""><figcaption></figcaption></figure>

После того как вы добавили необходимое количество категорий, можно приступать к созданию услуг.

## Создание услуг

Для создания услуги кликните по кнопке "Создать", которая расположена в правой части экрана:

<figure><img src="/files/RPkIi9pvkch8JnNnpPez" alt=""><figcaption></figcaption></figure>

Далее откроется модальное окошко со всеми настройками услуг:

<figure><img src="/files/vhNoSHNBCpcGK7g2pOeq" alt=""><figcaption></figcaption></figure>

Для начала выберите тип услуги "Перевозка":

<figure><img src="/files/Ush3hEkyldiRSpozuLKu" alt="" width="563"><figcaption></figcaption></figure>

После того как был выбран тип услуги "Перевозка", модальное окно с настройкой услуги приобретет следующий вид:

<figure><img src="/files/lQw8XGy7gNe1R1MkZueT" alt="" width="563"><figcaption></figcaption></figure>

1. **Название услуги**

<figure><img src="/files/KBKeorjhlDVpps6OwHfF" alt="" width="563"><figcaption></figcaption></figure>

Название услуги отображается в боте (в том числе с ИИ) и виджете.&#x20;

2. **Категория услуг**

<figure><img src="/files/YQhLTgCY9r0xjdkYS1Ia" alt="" width="563"><figcaption></figcaption></figure>

Поможет отнести создаваемую услугу к одной из категорий.&#x20;

3. **Вместимость**

<figure><img src="/files/qtwteGxvMW81D7p8hbxg" alt="" width="563"><figcaption></figcaption></figure>

Поможет указать количество мест для пассажирских перевозок.

{% hint style="info" %}
Можно указать любое необходимое количество
{% endhint %}

4. Стоимость услуги

<figure><img src="/files/FS4TGGdp21ImagSRHyUl" alt="" width="563"><figcaption></figcaption></figure>

Стоимость услуги рейса отображается в виджете онлайн-записи и в чат-боте в той валюте, которая была указана при создании филиала.&#x20;

5. Длительность услуги

<figure><img src="/files/ok3GEHwfY7y6WQ0McGpi" alt="" width="563"><figcaption></figcaption></figure>

Устанавливается в часах и минутах:

<figure><img src="/files/lAZiB9r7yOTjf7dBnrc0" alt="" width="325"><figcaption></figcaption></figure>

6. Описание услуги

<figure><img src="/files/I3qFUbIqYhN7nSmSVtgP" alt="" width="563"><figcaption></figcaption></figure>

Добавьте интересное и привлекающее описание услуги, а также укажите характеристики своей услуги.

7. Начальная и конечная остановка:

<figure><img src="/files/Mmm3BsIHoomANVyXzqda" alt="" width="563"><figcaption></figcaption></figure>

Начальная и конечная остановка будут отображены в виджете записи, а также в разделе услуги, чтобы вы могли видеть начальную и конечную точку отправления наглядно.

8. Тип услуги:

<figure><img src="/files/eSq6LKtZvuIUa3BoTwUg" alt="" width="563"><figcaption></figcaption></figure>

В заполненном виде услуга выглядит так:

<figure><img src="/files/XHI5HW1YoZrAttmV1KiX" alt="" width="563"><figcaption></figcaption></figure>

Нажмите "Добавить" услугу, чтобы сохранить настройки и добавить услугу в выбранную категорию.

Чтобы добавить еще услуги, нажмите кнопку "Создать услугу".

После того как вы создали необходимое количество услуг, вы можете развернуть категории и наглядно посмотреть созданные вами рейсы:

<figure><img src="/files/hpSXrBlyDEUd3UlxEX6Y" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Готово!&#x20;

Теперь вы знаете, как настроить услуги для онлайн-записи.&#x20;
{% endhint %}
