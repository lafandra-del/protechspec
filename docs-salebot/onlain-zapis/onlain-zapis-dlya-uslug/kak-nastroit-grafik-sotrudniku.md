# Как настроить график сотруднику

Чтобы просматривать записи и графики сотрудников в разделе "Календарь", необходимо задать рабочий график сотрудников.&#x20;

Для этого перейдите во вкладу "Рабочий график" в настройках сотрудника:

<figure><img src="/files/IpXuApLOMNfPnufMWshe" alt=""><figcaption></figcaption></figure>

Далее вы увидите **шаблон для автоматического заполнения графика** и расписание, в котором можно вручную проставить часы работы:

<figure><img src="/files/E0lHuhM71XqaJ2sUOTqb" alt=""><figcaption></figcaption></figure>

Чтобы выставить рабочий график на несколько недель вперед, заполните шаблон:

1. Расписание:

Задаются рабочие дни и выходные дни:

<figure><img src="/files/SY1tanEllXUynOp3q3vp" alt="" width="563"><figcaption></figcaption></figure>

2. Дата, с которой необходимо заполнить график:

<figure><img src="/files/mhblvFjNsoLMjzeJ53tD" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Если исполнитель работает с понедельника по пятницу (стандартизированная рабочая неделя), то необходимо задавать дату, начиная с понедельника.&#x20;
{% endhint %}

3. Рабочие часы

<figure><img src="/files/Tn1Ga8xwcz0ufHyeBNkg" alt="" width="563"><figcaption></figcaption></figure>

4. На сколько недель вперед выставить расписание:

<figure><img src="/files/VeOGxDVlS1FD35hq4Vcy" alt="" width="563"><figcaption></figcaption></figure>

Далее кликните на кнопку "Заполнить и сохранить график", тогда расписание выставится для конкретного сотрудника.&#x20;

Если необходимо добавить дополнительные рабочие часы или убрать рабочие часы, то можно воспользоваться расписанием, где вручную возможно убрать или добавить часы:

<figure><img src="/files/8thVcbXxf1rPSHBCNZUT" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Готово!&#x20;

Теперь вы знаете, как настроить услуги и исполнителей в разделе "Услуги".&#x20;
{% endhint %}
