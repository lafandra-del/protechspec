# Аналитика клиентов

Раздел "Клиенты" представлен в виде виджетов, установленных по умолчанию, а также конструктора виджета, где можно отобразить любые необходимые данные. Данный раздел состоит из следующих подразделов:

В разделе клиенты вы можете наблюдать сводку по рассылкам, отправленным и полученным сообщениям, а также соотношение клиентов в круговой диаграмме по подключенным каналам.&#x20;

### Переменные

Во вкладке "Переменные" необходимо указать значения переменной для учета статистики:

<div data-with-frame="true"><figure><img src="/files/PlMF3FM23Dhnwy3ODJmU" alt="" width="563"><figcaption></figcaption></figure></div>

На примере выше мы указали переменную age, в которую записывается возраст клиента в боте. Поскольку нами указано "По переменным клиента", система будет считывать значения клиентов именно по данной переменной и будет учитывать только тех, кому 18 и старше.&#x20;

В своем проекте вы можете использовать самые разные переменные, которые существуют в вашем проекте: это может быть и сумма оплат, например, если вы хотите высчитывать статистику по определенному тарифу; город проживания и т.п.&#x20;

{% hint style="warning" %}
Каждая строка в отображаемом виджете соответствует значению переменной, которую вы обозначили при его настройке.&#x20;

Максимальное количество строк (значений переменной) - 6.&#x20;
{% endhint %}

#### Операторы "И" и "ИЛИ"

<div data-with-frame="true"><figure><img src="/files/v3vnFlvmyOqvhI1ojVq2" alt="" width="442"><figcaption></figcaption></figure></div>

Если вы передаете для аналитики одно значение, то можно выбрать любой из операторов. Однако в случае, если передаете несколько значений, то стоит разобраться, в чем различие данных фильтров:

1. Фильтрация по оператору "И" обозначает включение значений всех переменных.&#x20;

То есть, если вы выбрали "И", то система будет осуществлять поиск и включение в статистику только тех, у кого есть <mark style="color:red;">**сразу несколько выбранных значений**</mark> переменных.&#x20;

Например, если вы выбрали переменную age, присвоили ей значений 18, а также выбрали переменную gender, присвоив ей значение male (мужчина), и поставили фильтрацию "И", то система осуществит выборку только по тем клиентам, которые являются мужчинами старше 18 лет.&#x20;

<div data-with-frame="true"><figure><img src="/files/PIg7pTbdjHYvgSpxUEEY" alt="" width="556"><figcaption></figcaption></figure></div>

Условно система понимает, что нужен клиент И старше 18, И мужчина. Других клиентов система не включит в статистику.&#x20;

**Как работает И:**

Можно также делать поиск только по одной переменной.

<div data-with-frame="true"><figure><img src="/files/Eyyk69uYLUBqGNwxjXve" alt="" width="483"><figcaption></figcaption></figure></div>

Если написать var1 = 1, var2 = 2, система вернет 0: это сделано для того, чтобы можно было ставить некоторые условия типа var1 != 1 **И** var1 > 0 - в таком случае система вернет тех клиентов, чьи переменные не равны одному и больше нулю (со значениями 2, 3, 4 и т.п.).&#x20;

{% hint style="warning" %}
Обращаем внимание!

Название переменной для **оператора И** для всех строчек должно быть одним.

Например:

per1 = 1 AND pre2 = 2 - является НЕверным!&#x20;

per1 < 400 AND per1 > 4 - верное написание для переменных.&#x20;
{% endhint %}

{% hint style="warning" %}
Если Вам необходимо использовать переменные следующим образом:

per1 = 1 AND pre2 = 2, то воспользуйтесь оператором ИЛИ.&#x20;
{% endhint %}

2. Фильтрация по оператору "ИЛИ" включает одно из несколько значений (или несколько).&#x20;

Такой фильтр применяется в том случае, если вам нужна выборка клиентов с различными значениями переменных: например, вам нужны клиенты либо старше 18 лет, либо проживающие в городе Москве.&#x20;

Тогда оператор "ИЛИ" включит в выборку всех, у кого есть одна из двух переменных age или city:

То есть, условно система понимает, что нужны те, кто ИЛИ старше 18, ИЛИ проживающие в Москве.&#x20;

**Как работает ИЛИ**:&#x20;

<div data-with-frame="true"><figure><img src="/files/SBoyhedRhmwpAt8oPpP6" alt="" width="483"><figcaption></figcaption></figure></div>

Переменные : var1 = 1, var2 = 2 , где система ищет любого клиента, у которого хоть одна из них совпадает с выбранным значением (*можно передать без значения - тогда система просто смотрит ее наличие*)

{% hint style="info" %}
Что такое переменные и как с ними работать, прочтите в статье "[Переменные](/chatbots/peremennye-i-funkcii/peremennye.md)".&#x20;
{% endhint %}

#### Конверсия

Чтобы рассчитать конверсию, воспользуйтесь одноименной вкладкой в конструкторе виджета:

<div data-with-frame="true"><figure><img src="/files/B9Ync6UP4czELZFhv9TH" alt="" width="563"><figcaption></figcaption></figure></div>

Поле "Что считать" поможет отобразит конверсию по 4 критериям:

1. **Количество** - подсчет числа клиентов за определенный период времени и по определенному событию/категории или сайту.

При подсчете количества можно выбрать, каким образом считать значения для клиентов:

a) все прохождения клиентов

b)  уникальные клиенты (т.е. впервые посетившие за определенный промежуток времени):

<div data-with-frame="true"><figure><img src="/files/NXNrENHpmNHAEG0JwHc0" alt=""><figcaption></figcaption></figure></div>

1. **Сумма значений** - система складывает количественные показатели значений, которые необходимо учесть для статистики;&#x20;
2. **Среднее значение** - сумма всех значений чисел, поделенных на их количество. То есть, например, было три выплаты: первая выплата - 1999, вторая выплата -2455, а третья -3455. Среднее значение будет равно 2 636 (сложили все три выплаты и поделили на их количество - три). &#x20;
3. **Медианное значение** - то число, которое является серединным значением для множества чисел: возьмем пример выплат с предыдущими значениями, где первая выплата 1999, вторая - 2455, а третья - 3455. Медианным значением в таком случае будет число 2455, стоящее строго среди множества чисел.&#x20;

Наглядно покажем разницу отображения виджета с разными значениями для поля "Что считать": возьмем в пример виджет:

<div data-with-frame="true"><figure><img src="/files/Gfd4dtpRaZPRre0XfbmM" alt="" width="562"><figcaption><p>"Что считать" ➡️ Количество</p></figcaption></figure></div>

В данном случае видно, что высчитывает система - это количество клиентов, которые купили/согласились купить шапки.&#x20;

Теперь изменим "Что считать" на сумму значений:

<div data-with-frame="true"><figure><img src="/files/ET1igXWTxjTMOTfb8kuP" alt="" width="560"><figcaption><p>"Что считать" ➡️ Сумма значений</p></figcaption></figure></div>

Теперь система высчитывает значения, присвоенные клиентам: то есть сколько всего шапок купили или согласились купить 10 клиентов.&#x20;

Далее поменяем "Что считать" на среднее значение:

<div data-with-frame="true"><figure><img src="/files/JWMPiyTeISDk9RQ0ejvc" alt="" width="563"><figcaption><p>"Что считать" ➡️ Среднее значение</p></figcaption></figure></div>

На примере выше видно, среднее значение количества купленных шапок на количество клиентов.

Далее рассмотрим разницу отображения количества и медианного значения:

<div><figure><img src="/files/rwvd8k6EBVUa7ciNYs2U" alt=""><figcaption><p>Количество</p></figcaption></figure> <figure><img src="/files/DH47vFhtiHCsCs3QaS4E" alt=""><figcaption><p>Медианное значение</p></figcaption></figure></div>

В обеих диаграммах видно итоговое количество, однако показатели в колонке клиентов разнятся: во втором случае отображается срединное, то есть медианное значение.

В поле "Как считать" вы выбираете подсчет процентов либо от первого шага либо от предыдущего. Вне зависимости от выбора, каким именно образом осуществлять подсчет, откроется настройка шага:

### Фильтры

Виджет в виде фильтра поможет отобразить данные, отфильтрованные определенным образом по установленным значениям.&#x20;

<div data-with-frame="true"><figure><img src="/files/7oWOLaFuQ5T2VEsJ2uVI" alt="" width="563"><figcaption></figcaption></figure></div>

Например, у вас есть несколько списков, в которые включены клиенты. Данный тип виджета поможет отобразить точное количество клиентов, которые присутствуют в каждом из списков и отобразит их на виджете.&#x20;

#### Как настраивать виджет?

Для начала необходимо добавить необходимое количество значений для отображения данных в виджете:

<div data-with-frame="true"><figure><img src="/files/6VtEAyKT237mG2lW2GU9" alt="" width="524"><figcaption></figcaption></figure></div>

{% hint style="success" %}
Всего виджет может принимать не более 6 значений.&#x20;
{% endhint %}

После этого выберите необходимый фильтр для отображения количества клиентов:

<div data-with-frame="true"><figure><img src="/files/5UB5lc5KUYw1z03rFd80" alt="" width="563"><figcaption></figcaption></figure></div>

Если вы не установите название шага, то по умолчанию он будет обозначен как "Шаг 1", "Шаг 2" и т.п.

Затем выберите необходимые фильтры для отображения:

1. Каналы - будет выведено количество клиентов из определенного канала.&#x20;

Например, в проекте у вас подключено несколько Телеграм-каналов: в фильтре можно отобразить значение для каждого.

<div><figure><img src="/files/19jpYPzoXM8PkcGnOgNn" alt=""><figcaption></figcaption></figure> <figure><img src="/files/M2kwgaJgjmFhxI4dyBJe" alt=""><figcaption></figcaption></figure> <figure><img src="/files/PkeboF6Gw4EkWeXcQY3k" alt=""><figcaption></figcaption></figure></div>

2. Списки и метки - значение отображает количество клиентов соответственно в определенных вами списках и метках.

<div data-with-frame="true"><figure><img src="/files/V1WC0pMlcmrX66TtPvUH" alt="" width="373"><figcaption></figcaption></figure></div>

Аналогично каналам, можно выбрать сразу несколько списков/меток для отображения клиента по фильтрам.

3. Блок воронки - шаг выразит количество клиентов по соответствующему (соответствующим) блокам воронки.&#x20;

<div data-with-frame="true"><figure><img src="/files/MfAd4mDyOkpBYrAZ697q" alt="" width="435"><figcaption></figcaption></figure></div>

4. По  клиентам: отписавшимся, с их исключением или отображением всех.

Можно применить как дополнительный фильтр в значении для клиентов: например, для просмотра данных по отписавшимся клиентам в определенных каналах.

{% hint style="danger" %}
Обращаем внимание!&#x20;

При использовании фильтра "Отображать только отписавшихся" будут отображаться клиенты, добавленные в указанный промежуток времени,  со статусом "Отписка" независимо от того, когда он отписался.
{% endhint %}

<div data-with-frame="true"><figure><img src="/files/RF3PEQa0U9FBAy764x4n" alt="" width="563"><figcaption><p>Отобразятся клиенты, добавленные со статусом "Отписка", незавимо от того, когда пользователь отписался.</p></figcaption></figure></div>

5. По тегу - системной переменной, присваиваемой клиенту.&#x20;
