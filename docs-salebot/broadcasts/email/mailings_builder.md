# Конструктор email-письма

Письма для email-рассылок можно собирать по вашему дизайну с помощью блоков.

<figure><img src="/files/MUGik51Cs5j5sDSrTKkY" alt="" width="563"><figcaption></figcaption></figure>

Тексты, кнопки, мессенджеры, изображения, а также инструменты для кастомизации блоков - это абсолютно новый уровень.

Забудьте о скучных шаблонных письмах - создавайте уникальность вместе с Salebot

## Секции (блоки) для письма

Конструктор письма можно найти во вкладке "Сообщение" при создании email-рассылки:

<figure><img src="/files/niYonCgrreuStwa5wM8L" alt="" width="563"><figcaption></figcaption></figure>

В конструкторе вы увидите пустой лист, в который можно вставить следующие блоки (секции):

1. Изображение
2. Текст
3. Пустой блок
4. Кнопка
5. Кнопки в социальные сети.&#x20;

Помимо этого блоки можно комбинировать друг с другом, добавляя, например, в одну секцию несколько элементов письма.&#x20;

<figure><img src="/files/KvYNJQQIZ09Kkjj8MAgA" alt=""><figcaption></figcaption></figure>

В блоках существуют общие настройки отображения секции:&#x20;

1. Внутренние отступы;
2. Рамка;
3. Скругление рамки;&#x20;
4. Фон.

<figure><img src="/files/8Z0k3YuF8SRT7O85VyfT" alt="" width="375"><figcaption></figcaption></figure>

Давайте для начала разберем настройки каждой секции, а также на примере мы покажем, как можно кастомизировать несколько различных блоков в одной секции.&#x20;

### Изображение

Чтобы добавить изображение, обратитесь к меню в правой области экрана. Там вы найдете иконку, по нажатию на которую вы сможете добавить изображение:

<figure><img src="/files/6sxLeo7aZY5ai99o9Zck" alt="" width="563"><figcaption></figcaption></figure>

При клике на кнопку вы увидите, что секция была добавлена в письмо:

<figure><img src="/files/ylyRFGIhn13Yht7sr5o9" alt=""><figcaption></figcaption></figure>

Чтобы добавить изображение, перейдите в соответствующую вкладку и загрузите необходимый вам файл либо с вашего устройства:

<figure><img src="/files/vGpWm5ljRb5pi7K6pX4D" alt=""><figcaption></figcaption></figure>

Далее вы увидите, что изображение было добавлено в секцию:

<figure><img src="/files/O0l29KdYWQRtgG4hgzWx" alt=""><figcaption></figcaption></figure>

Из изображения можно сделать подобие кнопки, по клику на которую в письме, получатели рассылки смогут перейти на какую-либо необходимую страницу.&#x20;

Для этого достаточно указать ссылку необходимой страницы в данном поле:

<figure><img src="/files/gzxj3G3GABfSsFOJvXBd" alt="" width="563"><figcaption></figcaption></figure>

### Текст

Блок текст можно добавить как внутрь секции с изображением (и любым другим блоком), так и в отдельно.&#x20;

{% hint style="warning" %}
Внимание!

Изменение шрифтов недоступно.&#x20;
{% endhint %}

1. Чтобы добавить текст внутрь секции, например, с изображением, наведите на блок с изображением (или любым другим блоком справа) и найдите иконку "+":

<figure><img src="/files/e5X9kscyHovAZxzHbXLS" alt="" width="284"><figcaption></figcaption></figure>

Далее кликните на плюс, после чего откроется выпадающий список с блоками:

<figure><img src="/files/nHgNBKgn1hEhJu81IX8k" alt="" width="259"><figcaption></figcaption></figure>

Как видно, текстовый блок добавился внутрь секции, в котором уже лежит изображение:

<figure><img src="/files/OzWjdphGxVHzHO2LxRjz" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Обращаем внимание!

В одну секцию вы можете добавить четыре элемента (блока) с контентом.
{% endhint %}

2. Чтобы добавить отдельную секцию в письме с текстовым блоком (или любым другим) наведите на левую часть экрана и найдите в быстрых действиях иконку "+":

<figure><img src="/files/t1qr9iDRELeMSQhekAEZ" alt="" width="563"><figcaption></figcaption></figure>

Далее нажмите на плюс и выберите секцию, которую необходимо добавить:

<figure><img src="/files/v86RBLyGQmri3Dkz0W5o" alt="" width="375"><figcaption></figcaption></figure>

Тогда добавится новая секция с текстовым блоком:

<figure><img src="/files/vDh2QPpNBshHyfw3cuIr" alt="" width="563"><figcaption></figcaption></figure>

Для редактирования кликните по тексту, тогда сверху вы увидите меню с настройками, аналогичными для любых текстовых редакторов:

<figure><img src="/files/gXRjijXTXv675FttrCNN" alt=""><figcaption></figcaption></figure>

В меню сверху блока с текстом можно установить жирность, курсив, подчеркивание:

<figure><img src="/files/4Z65jyuKPR32xLMucMzh" alt=""><figcaption></figcaption></figure>

А также оформить текст списком, добавить ссылку и выровнять относительно правого, левого края или середины.

Чтобы настроить размер начертания текста, высоту шрифта, интервал и цвет перейдите в меню настроек блока "Текст", расположенный в меню справа:

<figure><img src="/files/oYE9Zzqjb5URurM3dXkn" alt=""><figcaption></figcaption></figure>

### Кнопка&#x20;

Блок с кнопкой добавляется аналогично любому блоку: его можно установить как отдельной секцией, так и внутрь секции с уже содержащимися в ней элементами:

<figure><img src="/files/WxxgNRADD4CRmALDQF17" alt=""><figcaption></figcaption></figure>

В кнопку можно вставить ссылку на любую необходимую страницу:

<figure><img src="/files/FKdaHrsFHqTYKCS9M5sW" alt=""><figcaption></figcaption></figure>

Чтобы изменить текст, кликните по текстовому полю внутри кнопки:

<figure><img src="/files/8l81hPLD9Y1J9Uu05AFf" alt="" width="338"><figcaption></figcaption></figure>

Затем пропишите необходимый текст:

<figure><img src="/files/MIxcxUiQLkOmQX71S9g6" alt="" width="425"><figcaption></figcaption></figure>

Чтобы изменить размер кнопки, найдите поле со стрелками в меню:

<figure><img src="/files/d3FjZ80YbcSgU8vQzrUJ" alt="" width="375"><figcaption></figcaption></figure>

Далее установите необходимый размер, вводя числовое значение: поле принимает значение в пикселях.&#x20;

Чтобы изменить расположение кнопки относительно центра, правого или левого края, воспользуйтесь данными настройками:

<figure><img src="/files/nKq5iaVpqOLIiIDv8nEf" alt="" width="563"><figcaption></figcaption></figure>

Также в меню существует поле для изменения цвета кнопки:

<figure><img src="/files/hh8xNvcRrpnGb07tdQx7" alt="" width="465"><figcaption></figcaption></figure>

### Кнопки "Социальные сети"

В письме вы можете установить также кнопки, которые перенаправят получателей ваших писем в социальные сети, которые подключены к вашему проекту:

<figure><img src="/files/CXadc6xypCxoQKWKvEng" alt=""><figcaption></figcaption></figure>

В настройках блока можно выбрать, на какой именно канал вы хотите перенаправлять ваших клиентов:

<figure><img src="/files/FhzlsRmpRAnMOznskvH2" alt="" width="563"><figcaption></figcaption></figure>

Также в меню настроек существует настройка стиля для кнопок социальных сетей:

<figure><img src="/files/aDKdjAb5zTE3u2ILe1XW" alt="" width="563"><figcaption></figcaption></figure>

1. Стандартные: принимают оригинальный стиль иконок мессенджеров:

<figure><img src="/files/9OB3kknWu3hyw9MOq8CQ" alt="" width="326"><figcaption></figcaption></figure>

2. С рамкой: прозрачные иконки, цвет рамки которых выдержен в стандартных цветах для данного мессенджера:

<figure><img src="/files/xnK74EUVN5Q7PIl1nUvM" alt="" width="371"><figcaption></figcaption></figure>

3. Моноколор: иконки выдержаны в одном цветовом решении:

<figure><img src="/files/qdQfr9P74oPp5h8f7ZG8" alt="" width="365"><figcaption></figcaption></figure>

Чтобы скруглить углы иконок, введите в поле "Скругление" необходимое числовое значение в пикселях:

<figure><img src="/files/Hs41t0IA9YUmbAC32A9u" alt=""><figcaption></figcaption></figure>

## Настройки блоков внутри секции

Для каждого элемента внутри секции существуют одинаковые настройки отступов, рамки и фона.&#x20;

Можно настроить блок отдельно от каждого элемента, а также выбрать

### Внутренние отступы элемента

Внутренние отступы для элемента можно настроить в соответствующих полях:

<figure><img src="/files/POkPbp84Yvhm6yvRhkBA" alt=""><figcaption></figcaption></figure>

Данные отступы относятся к расположению именно блока с контентом, относительно его рамки.&#x20;

### Рамка (внутренняя)

В секции существует две рамки, которые можно кастомизировать: внешняя (для всей секции) и внутренняя (для изображения или иных элементов)

{% hint style="info" %}
Как редактировать внешнюю рамку секции, рассказали в разделе "Общие настройки секций" -> "[Рамка](#ramka)"
{% endhint %}

Внутренню рамку, обрамляющую блок с контентом, можно кастомизировать с использованием любого цвета, толщины, выбирая также необходимые для выделения границы и вид линии:

<figure><img src="/files/kG2cBGI4xnrY14tLulmj" alt="" width="375"><figcaption></figcaption></figure>

При этом настройки каждой грани рамки можно кастомизировать отдельно от других сторон.

Для этого кликните по необходимой стороне рамки:

<figure><img src="/files/zTANSFli0IGNZf6EqLOy" alt="" width="563"><figcaption></figcaption></figure>

А затем выберите цвет, толщину и начертание рамки.&#x20;

Чтобы кастомизировать рамку с четырех сторон, кликните по соответствующей иконке:

<figure><img src="/files/1vMsVqYTQ2wO3k18mfEw" alt="" width="563"><figcaption></figcaption></figure>

#### Скругление внутренней рамки

Для внутренней рамки также можно установить скругление углов в px:

<figure><img src="/files/aLPrfni62iwbLBjyMVgf" alt="" width="375"><figcaption></figcaption></figure>

Поля для каждого из углов рамки принимают числовое обозначение в пикселях.&#x20;

### Фон блока с контентом внутри секции

## Настройки секции

Секцию с элементами контента можно настроить отличительно от иных секций письма, используя цветовые решения для фона, добавив рамки и отступы.&#x20;

### Отступы

Отступы секции применяются ко всем элементам, находящимся внутри секции:

<figure><img src="/files/DOk64jVShSMuewgBVD5O" alt=""><figcaption></figcaption></figure>

Для того чтобы установить отступы для блоков относительно внешней рамки секции, необходимо потянуть за края рамки, изменяя положение элементов относительно верхнего и нижнего, а также правого и левого края:

<figure><img src="/files/Vjs5arkQfd5ZxPu0xlYF" alt="" width="563"><figcaption></figcaption></figure>

Либо можно установить числовые значения в полях, установленных в правом меню в разделе "Секция":
